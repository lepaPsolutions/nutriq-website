# Nutriq — Marketing Site

The public marketing site for [nutriq.gr](https://nutriq.gr) — nutrition software for dietitians.

Separate from the app (Next.js, hosted on AWS). This is a static Astro site optimized for SEO and performance.

## Stack

- **Astro 5** — static site generator
- **Tailwind CSS** — utility-first styling
- **TypeScript** — type-safe i18n
- **Inter** (Greek + Latin Extended) — single font, professional readability in both languages
- **Vercel** — deployment target

## Run locally

```bash
npm install
npm run dev
```

Open <http://localhost:4321>

## Build for production

```bash
npm run build
npm run preview
```

Output goes to `dist/`.

## Project structure

```
src/
├── components/        Reusable Astro components
│   ├── home/         Home page sections
│   ├── pricing/      Pricing matrix
│   └── ...
├── i18n/             Translation system
│   ├── ui.ts        All UI copy (EL + EN)
│   ├── pricing.ts   Pricing plans source of truth
│   └── utils.ts     Helper functions
├── layouts/          Page layouts
├── pages/            File-based routing
│   ├── *.astro      Greek pages (default lang at root)
│   └── en/          English mirror
├── styles/
│   └── global.css   Tailwind layers + custom utilities
└── env.d.ts
```

## Adding a new page

1. Create `src/pages/your-page.astro` (Greek)
2. Mirror at `src/pages/en/your-page.astro` (English)
3. Add nav entry to `src/components/Header.astro` if needed
4. Add translations to `src/i18n/ui.ts`

## i18n

- Default language: **Greek** at root (`/`)
- English: `/en/` prefix
- All copy lives in `src/i18n/ui.ts`
- Use `useTranslations(lang)` in Astro frontmatter

## Brand tokens

Defined in `tailwind.config.mjs`:

- `brand` (teal): `#0AA6A9` accent, `#00494D` deep (CTAs)
- `peach`: `#F7AE7E` (accent, sparing use)
- `cream`: warm neutrals for section breaks
- `gray`: neutral SaaS scale
- Font: Inter (display + body, weight hierarchy)

## Deploy

Connected to Vercel. Push to `main` → auto-deploy production. Push to any other branch → preview deploy.

Headers, redirects, and caching configured in `vercel.json`.

## SEO

- Sitemap auto-generated by `@astrojs/sitemap`
- Per-page meta tags + Open Graph in `BaseLayout.astro`
- `hreflang` tags for EL ↔ EN bilingual SEO
- Organization + FAQPage structured data (JSON-LD)
- `robots.txt` permissive
